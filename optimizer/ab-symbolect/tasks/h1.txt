Add and export a function toCSV(rows) in src/index.js that serializes an array of row objects into an RFC 4180 CSV string — the inverse of parseCSV. Use the first row's keys as the header row. A field must be wrapped in double quotes if it contains a comma, a double-quote, or a newline, and any embedded double-quote is escaped by doubling it. Rows are joined by a single newline with no trailing newline; an empty array yields an empty string. It will be graded by round-tripping randomly generated rows through a strict CSV parser, so every field value — including ones containing commas, double-quotes, newlines, leading/trailing spaces, and unicode — must survive the round trip. Handle all of it, not just simple cases.
